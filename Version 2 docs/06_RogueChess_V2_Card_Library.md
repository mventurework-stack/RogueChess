# Rogue Chess V2 -- Card Library

## Structure

Each card should contain: - Name - Scrap Cost - Category - Effect -
Target - Counterplay

## Example Cards

### Sprint

Cost: 2 Category: Movement Effect: +1 movement this turn.

### Repair Kit

Cost: 2 Category: Repair Effect: Restore health.

### Shield Matrix

Cost: 3 Category: Defense Effect: Ignore next damage this turn.
